# E-Commerce Frontend

A modern e-commerce frontend built with **Next.js** and connected to a custom Ecommerce API.  
This project focuses on delivering a clean, responsive, and user-friendly shopping experience.

---

##  Live Demo

 https://e-commerce-frontend-lemon-seven.vercel.app/products

---

##  Features

- Responsive UI
- Product listing page
- Product detail pages
- Shopping cart functionality
- Dynamic routing with Next.js
- API integration
- Clean modern design
- Fast frontend performance

---

##  Tech Stack

### Frontend
- Next.js
- React
- JavaScript / TypeScript
- Tailwind CSS
- Fetch API / Axios

### Deployment
- Vercel

### Backend
- Custom Ecommerce REST API
